## 词条系统规则（2026年3月6日更新）

### ✅ 词条定义
- 词条类似《遮天》中的**体质**，有独特的外观和特性
- 不是简单的属性加成，而是完整的体质系统

### ✅ 寿命损耗阶梯
| 等级 | 消耗寿命 |
|------|----------|
| F级 | 1年 |
| E级 | 2年 |
| D级 | 5年 |
| C级 | 10年 |
| B级 | 20年 |
| A级 | 50年 |
| S级 | 100年 |
| SS级 | 300年 |
| SSS级 | 500年 |

### ✅ 寿命增加途径
- 境界突破：每个大境界增加100-1000年
- 天材地宝：蟠桃、人参果等增加50-500年
- 特殊功法：如《玄黄经》每年增加10年
- 奇遇：随机获得50-1000年

### ✅ 必须遵守的规则
1. **词条掠夺**：可以从人类/妖兽身上掠夺词条
2. **每3章最多1个D级或以上词条**：第4章属于新的周期
3. **状态必须明确列出**：每章开始和结束必须明确写出当前词条
4. **寿命消耗必须追踪**：每次掠夺都要扣除寿命并记录

### ❌ 绝对禁止的错误
1. **禁止词条数量错误**：必须精确计算当前有几个词条
2. **禁止寿命消耗不清**：每次扣除必须明确说明
3. **禁止境界跳跃**：必须从上一章结尾状态承接
